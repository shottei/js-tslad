## Commands List
*ping* | **Shows latency of bot** (API latency not shown in command!)

*botinvite* | **Gives invite to TsLad**

*ban* | **Bans user from the server** (Perma Bans Only!)

*kick* | **Kicks user from the server** (No Looped / Timed Kicks!)
